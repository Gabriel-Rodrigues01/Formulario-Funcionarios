# Formulário de Cadastro de Funcionários

Um projeto simples para gerenciar o cadastro de funcionários. A aplicação permite adicionar, visualizar, editar e excluir registros de funcionários, tudo rodando diretamente no navegador.

## 🚀 Funcionalidades

* **Cadastrar:** Adicionar novos funcionários ao sistema através de um formulário.
* **Listar:** Visualizar todos os funcionários cadastrados em uma tabela.
* **Editar:** Atualizar as informações de um funcionário existente.
* **Excluir:** Remover um funcionário do sistema.

*(Observação: Se seu projeto ainda não tiver todas essas funções, sinta-se à vontade para remover os itens da lista acima.)*

## 💻 Tecnologias Utilizadas

Este projeto foi construído utilizando tecnologias web front-end:

* **HTML5:** Para a estrutura e o conteúdo da página.
* **CSS3:** Para a estilização e o design.
* **JavaScript (JS):** Para a interatividade, manipulação do formulário e gerenciamento dos dados (provavelmente usando `localStorage`).

## 📁 Como Executar o Projeto

Como este é um projeto front-end puro, você não precisa de um servidor.

1.  Clone este repositório (ou baixe os arquivos ZIP).
2.  Navegue até a pasta do projeto.
3.  Abra o arquivo `index.html` no seu navegador web de preferência.

E pronto! A aplicação estará funcionando.
